# Oficina-de-jogos-CEFET-RJ
Repositório destinado a mostrar os materiais desenvolvidos durante minha participação na oficina de jogos feita no CEFET-RJ no laboratório MULTISENS, no qual desenvolvi um protótipo de um Jogo de luta com os outros participantes na Unreal Engine. Consegui desenvolver minhas habilidades técnicas, artísticas e softskills como o trabalho em equipe. Nosso objetivo era criar um protótipo para que ele fosse apresentado na SEPEX (semana de projetos e extenções no CEFET_RJ) e assim o fizemos, apresentando também os outros projetos do laboratório MULTISENS.
